# 文件共享系统

一个功能完整的局域网文件共享工具，通过Web浏览器轻松访问和下载共享文件夹中的文件。

## 功能亮点

### 🖥️ 现代化启动器界面
- **ttkbootstrap Minty 主题**，清新薄荷绿配色
- 支持5种主题切换（Minty、Flatly、Simplex、Litera、Darkly）
- 历史路径记录，快速选择常用共享文件夹
- 端口检测功能，自动查找可用端口
- 本机IP自动检测，一键复制访问链接
- 图形化配置，操作简单直观

### 🌐 Web界面访问
- 通过浏览器直接访问共享文件夹，无需安装客户端
- 支持HTTP协议，兼容所有主流浏览器
- 跨平台支持，Windows、Linux、macOS均可使用

### 📁 文件管理
- 文件列表展示，清晰直观
- 支持按名称、大小、修改日期排序
- 文件下载功能，一键下载所需文件
- 文件大小和修改时间显示
- 文件类型图标识别
- 完美支持中文文件名显示
- 返回上一级按钮，方便目录导航

### 📤 文件上传
- 支持文件上传到当前浏览目录
- 拖放上传，操作便捷
- 实时上传进度显示
- 支持批量上传

- 暂停/恢复上传功能
- 完美支持中文文件名，保留原始文件名

- 文件夹项目计数显示

## 快速开始

### 方式一：使用打包的exe文件（推荐）
1. 双击运行 `文件共享.exe`
2. 选择要共享的文件夹
3. 设置管理员密码（可选）
4. 点击启动服务器
5. 在浏览器中访问显示的URL

### 方式二：从源代码运行
1. 安装Python 3.8及以上版本
2. 安装依赖：`pip install -r requirements.txt`
3. 运行启动器：`python launcher.py`
4. 选择要共享的文件夹
5. 设置管理员密码（可选）
6. 点击启动服务器
7. 在浏览器中访问显示的URL
