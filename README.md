# 局域网文件共享系统

一个功能完整的局域网文件共享工具，通过Web浏览器轻松访问和下载共享文件夹中的文件。

## 功能亮点

### 🌐 Web界面访问
- 通过浏览器直接访问共享文件夹，无需安装客户端
- 支持HTTP协议，兼容所有主流浏览器
- 跨平台支持，Windows、Linux、macOS均可使用

### 📁 文件管理
- 文件列表展示，清晰直观
- 支持按名称、大小、修改日期排序
- 多层文件夹浏览，轻松导航
- 文件下载功能，一键下载所需文件

### 🎨 现代化界面
- 现代化扁平设计风格
- 响应式布局，完美适配桌面和移动设备
- 流畅的交互体验
- 美观的文件图标显示

### 🌏 中文支持
- 完美支持中文文件名和路径
- UTF-8编码，确保中文正确显示
- 无乱码问题

### 🔒 安全可靠
- 路径安全验证，防止目录遍历攻击
- 只读访问，保护文件安全
- 文件大小限制，防止大文件占用带宽
- 权限控制，确保访问安全

### 🚀 易于使用
- 图形化界面，操作简单直观
- 一键启动，无需复杂配置
- 自动显示本机IP和访问URL
- 支持系统托盘图标（Windows）

### 📱 移动端优化
- 响应式设计，手机和平板完美适配
- 触控优化，操作流畅
- 自适应布局，各种屏幕尺寸都能良好显示

### ⚡ 高性能
- 快速响应，即时加载
- 高效文件传输
- 低资源占用

### 🎯 核心功能
- Web服务器功能
- 文件列表展示
- 文件下载
- 目录导航
- 文件排序
- 中文文件名支持
- 错误处理
- IP地址显示
- 系统托盘集成

### 💾 大文件支持
- 支持下载超大文件
- 最大支持10GB文件下载
- 稳定的文件传输
- 断点续传支持

## 使用场景

- 家庭网络文件共享
- 办公室内部文件分发
- 临时文件传输
- 局域网资源访问
- 多设备文件同步
- 大文件快速传输

## 系统要求

- Windows 10及以上
- 支持局域网环境
- 现代浏览器（Chrome、Firefox、Edge、Safari等）

## 快速开始

1. 双击运行程序
2. 选择要共享的文件夹
3. 点击启动服务器
4. 在浏览器中访问显示的URL
